### Installation guide

```bash
# clone our repo
$ git clone https://github.com/nenadpnc/rock-paper-scissors.git game

# change directory to your app
$ cd game

# install the dependencies with npm
$ npm install

# start the server
$ npm start
```
go to [http://localhost:8080](http://localhost:8080) in your browser.
